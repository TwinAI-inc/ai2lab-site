PMF

If I roll a die, what's the probability of getting exactly a 4? How would you express that as a function?

Probability Mass Function

Gives the probability that a discrete random variable equals a specific value.

$$P(X = x)$$ for each value of $x$
RULES
1. $0 \le P(X{=}x) \le 1$ for all $x$
2. $\sum_{\text{all } x} P(X{=}x) = 1$
EXAMPLE: ROLL A FAIR DIE

Let $X \triangleq$ outcome of rolling a fair die. $\;x \in \{1, 2, 3, 4, 5, 6\}$

Probability Density Function

11 / 34Prof. Mohammad DehghaniNortheastern University
CONTINUOUS DISTRIBUTIONS

PDF

What's the probability that your Uber arrives in exactly 4.000... minutes? Why is that a weird question?

Probability Density Function

KEY IDEA

For continuous variables, P(X = exact value) = 0. Always.

Instead, we find probability of a range:

$$P(a \le X \le b) = \int_a^b f(x)\,dx$$
RULES
1. $f(x) \ge 0$ everywhere
2. $\int_{-\infty}^{\infty} f(x)\,dx = 1$
3. $P(X = \text{exact}) = 0$

PMF = heights of bars.   PDF = area under the curve.

That's the key difference. Discrete counts. Continuous measures.

APPLIED EXAMPLE

Amazon Delivery Time Is Continuous

Amazon promises delivery between 2-5 days. Let X $\triangleq$ delivery time (days). Every delivery time between 2 and 5 days is equally likely. $\;x \in [2, 5]$

THE PDF
$$f(x) = \frac{1}{5-2} = \frac{1}{3}$$
THE MATH
Q1: P(X = 3 exactly)?
$= 0$ (continuous — no mass at a point)
Q2: $P(2 \le X \le 3) = ?$
$$= (3-2) \times \frac{1}{3} = \boxed{\frac{1}{3}}\approx 0.333$$
Q3: $P(X > 4) = ?$
$$= (5-4) \times \frac{1}{3} = \boxed{\frac{1}{3}}\approx 0.333$$
IDENTIFY THE DISTRIBUTION
$X \sim \text{Uniform}(a{=}2,\; b{=}5)$
Why? All values in [2, 5] are equally likely — constant density $f(x) = \frac{1}{b-a}$.

PDF — INTEGRATION

Call Center: Wait Time f(x) = 2x

Let X $\triangleq$ wait time (minutes). $\;x \in [0, 1]$. The PDF is: $f(x) = 2x$ for $0 \le x \le 1$, and 0 otherwise.

THE GRAPH

f(x) = 2x is a straight line from (0,0) to (1,2)

THE MATH
Q1: Verify it's a valid PDF
$\int_0^1 2x\,dx = [x^2]_0^1 = 1 - 0 = \mathbf{1}$ ✓
Q2: P(wait > 0.5 min)?
$\int_{0.5}^1 2x\,dx = [x^2]_{0.5}^1 = 1 - 0.25 = \boxed{0.75}$
Q3: E[X] = ?
$\int_0^1 x \cdot 2x\,dx = \left[\frac{2x^3}{3}\right]_0^1 = \boxed{\frac{2}{3}}\approx 40\text{s}$

CDF

Cumulative Distribution Function

17 / 34Prof. Mohammad DehghaniNortheastern University
CUMULATIVE

CDF: Cumulative Distribution Function

$$F(x) = P(X \le x)$$

"What's the probability that X is at most x?"

DISCRETE
$$F(x) = \sum_{k \le x} P(X = k)$$
Step function — jumps at each value
Add up all probabilities up to $x$
CONTINUOUS
$$F(x) = \int_{-\infty}^{x} f(t)\,dt$$
Smooth curve from 0 to 1
Area under PDF up to $x$
Always increasing
0 to 1, never decreases
Key trick
$P(a < X \le b) = F(b) - F(a)$
Boundaries
$F(-\infty)=0$, $F(\infty)=1$

CDF — CONTINUOUS EXAMPLE

Battery Life: $f(x) = 2x$ on [0,1]

Let X ≜ battery life (hours, scaled). PDF: $f(x) = 2x$ for $0 \le x \le 1$.

FIND THE CDF
$F(x) = \int_0^x 2t\,dt$
$$F(x) = [t^2]_0^x = x^2$$

So $F(x) = x^2$ for $0 \le x \le 1$

Check: $F(0)=0$, $F(1)=1$ ✓

CRUNCH IT
Q1: $P(X \le 0.5) = ?$
$F(0.5) = 0.5^2 = \boxed{0.25}$
Q2: $P(0.5 < X \le 0.8) = ?$
$$= F(0.8)-F(0.5) = 0.64-0.25 = \boxed{0.39}$$

CHALLENGE

What is the Inverse of a CDF?

CDF: "Given $x$, what's the probability?" Inverse: "Given a probability, what's $x$?"

SVG: CDF vs INVERSE
CDF (forward)

$$F(3) = P(X \le 3) = 0.75$$

INVERSE CDF

$$F^{-1}(0.75) = 3$$

REAL USE: PERCENTILES

95th percentile of exam scores = $F^{-1}(0.95)$
Doctor says "90th percentile cholesterol" = inverse CDF

THE MOST IMPORTANT NUMBER

Expected Value

If you roll a die 1,000 times, what do you expect the average roll to be? Why?

If you repeated this experiment forever, what would the average outcome be?

DISCRETE
$$E[X] = \sum_x x \cdot P(X{=}x)$$

Multiply each value by its probability, then sum

CONTINUOUS
$$E[X] = \int_{-\infty}^{\infty} x \cdot f(x)\,dx$$

Same idea, but integrate instead of sum

Intuition: E[X] is the "balance point" of the distribution. Where it would balance on a seesaw.

INTERPRETATION

What E[X] Really Means

Balance Point

If you placed the distribution on a seesaw, $E[X]$ is where it balances perfectly.

Long-Run Average

Repeat the experiment 10,000 times. The average of all outcomes converges to $E[X]$.

Fair Price

In gambling/insurance, $E[X]$ is the "fair" amount to charge — the break-even point.

Key insight: $E[X]$ is NOT the most likely outcome. A die has $E[X] = 3.5$ — but you can never roll 3.5.

DID YOU KNOW?

The House Always Wins

Every casino game is designed so that E[X] < 0 for the player. In American roulette, E[X] = -$0.053 per dollar bet. Over millions of spins, the casino is mathematically guaranteed to profit. Insurance companies use the exact same math — they set premiums so E[payout] < E[premium].

Casino Design Insurance Pricing Stock Valuation Warranty Pricing Decision Analysis

SPREAD

Variance & Standard Deviation

E[X] tells you the center. Variance tells you how spread out things are.

VARIANCE
$$\text{Var}(X) = E[(X - \mu)^2]$$

Average squared distance from the mean

STANDARD DEVIATION
$$\sigma = \sqrt{\text{Var}(X)}$$

Same units as X — easier to interpret

Two factories both produce bolts averaging 10mm. Factory A: σ=0.1mm. Factory B: σ=2mm. Which would you trust?

Two Coffee Machines: Which Is Better?

Both machines target 8 oz per cup. Let $X_A \triangleq$ oz poured by Machine A, $X_B \triangleq$ oz poured by Machine B. $\;x \in (0, \infty)$

MACHINE A
$E[X_A] = 8$ oz

MACHINE B

QUICK REFERENCE

Notation Guide

UPPERCASE = FUNCTION
$X$ — random variable (uppercase)
$F(x)$ — CDF (uppercase F)
$P(X = x)$ — probability (uppercase P)
$E[X]$ — expected value (uppercase E)
LOWERCASE = VALUE / DENSITY
$x$ — a specific value (lowercase)
$f(x)$ — PDF / PMF (lowercase f)
$p$ — probability parameter
$\mu, \sigma$ — mean, std dev
KEY RELATIONSHIPS
$f(x)$ (density) $\xrightarrow{\text{integrate}}$ $F(x)$ (cumulative) $\xrightarrow{\text{invert}}$ $F^{-1}(p)$ (quantile)
